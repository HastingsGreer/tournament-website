Bracket Buster: A Comp Sci Project

Bracket Buster is a web application to track team records for small sports leagues.

Bracket Buster allows league organizers to track wins and losses for their teams. It also allows players and fans to see who is leading the league.

Website Structure:
  • index.html :
    main page.
    list links to ongoing tournaments
    If logged in:
    • List links to tournaments that you manage
    • Create tournament link
    If not logged in:
    • Login and create user panels
  • view_tournament.html
    just shows the tournament so far.
    this page is dynamically generated for each tournament that has been created
    if logged in:
    --next to each undecided game there are two buttons so you can tell the website who won that game
  • create_tournament.html:
    Select N number of competitors (must be a power of two)
    N textboxes for team names (each initialized to "bye")
    Create tournament button: this goes to the view_tournament.html for the tournament you just made
