Bracket Buster: A Comp Sci Project

Bracket Buster is a web application to manage small sports tournaments. It allows league organizers to track wins/losses for their teams and allows players and fans to see who is up next to play and who is leading the league.

Website Structure:
  • Main Page Mockup lists links to ongoing tournaments.
    • If logged in:
      • List links to tournaments that you manage
      • Create tournament link
    • If not logged in:
      • Login and create user panels
  • view_tournament.html shows the tournament so far and is dynamically generated for each tournament that has been created.
    • If logged in:
      • Next to each undecided game there are two buttons so you can tell the website who won that game
  • Create Tournament Page has the ability to select N number of competitors (must be a power of two).
    • There are N textboxes designated for team names (each initialized to "bye")
    • Create tournament button goes to the view_tournament.html for the tournament you just made
